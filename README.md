# 💰 Finance Tracker (MERN Stack)

A full-stack Finance Tracking application built with the MERN (MongoDB, Express.js, React.js, Node.js) stack. This app allows users to manage their personal finances, track incomes and expenses, visualize data using charts, and maintain financial discipline.

---

## 🚀 Features

- User Authentication (JWT-based)
- Add, Edit, Delete Transactions (Income & Expense)
- Filter transactions by date, type, and category
- Dashboard with analytics and chart visualization (Pie, Bar, Line)
- Secure backend API using Node.js and Express.js
- MongoDB for data storage
- Responsive and modern UI with React.js + Tailwind CSS
- Protected Routes & Logout functionality

---

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS, Axios, Chart.js / Recharts
- **Backend**: Node.js, Express.js, MongoDB, Mongoose
- **Authentication**: JSON Web Tokens (JWT), bcryptjs
- **Deployment**: Vercel (frontend) + Render/Railway (backend)
